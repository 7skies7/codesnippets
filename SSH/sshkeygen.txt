==================================================
1) Add ssh keygen
==================================================

	$ ssh-keygen -t rsa

	Adds a private/public rsa key pair.

==================================================
2) Add ssh keygen (private)
==================================================

	$ ssh-keygen -t rsa -b 4096 -C "ext.pmudliyar@cma-cgm.com"

	Adds a private/public rsa key pair.
